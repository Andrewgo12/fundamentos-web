Calculadoras de Campos Eléctricos

Fuerza Eléctrica entre dos Cargas F = k * (q1 * q2) / r²

Campo Eléctrico de una Carga E = k * Q / r²

Potencial Eléctrico de una Carga V = k * Q / r

Capacitancia de un Condensador C = Q / V

Energía Almacenada en un Condensador U = 0.5 * C * V²

Campo Eléctrico entre dos Cargas E = k * (q1 / r²) + k * (q2 / r²)

Potencial Eléctrico entre dos Cargas V = k * (q1 / r) + k * (q2 / r)

Calculadoras de Electricidad

Ley de Ohm V = I * R

Potencia Eléctrica P = V * I

Resistencia en Serie R_total = R1 + R2 + ... + Rn

Resistencia en Paralelo 1 / R_total = 1 / R1 + 1 / R2 + ... + 1 / Rn

Capacitancia de un Condensador en Serie 1 / C_total = 1 / C1 + 1 / C2 + ... + 1 / Cn

Capacitancia de un Condensador en Paralelo C_total = C1 + C2 + ... + Cn

Calculadoras de MRUA

Velocidad Final v_f = v_i + a * t

Distancia Recorrida d = v_i * t + 0.5 * a * t²

Aceleración a = (v_f - v_i) / t

Velocidad Media v_m = (v_i + v_f) / 2

Calculadoras de Movimiento Circular

Velocidad Tangencial v = ω * r

Aceleración Centripeta a_c = v² / r

Periodo de Rotación T = 2 * π * r / v

Frecuencia f = 1 / T

Fuerza Magnética F = q * v * B * sen(θ)

Campo Magnético por una Corriente B = μ₀ * I / (2 * π * r)

Fuerza sobre un Hilo con Corriente F = I * L * B * sen(θ)

Ley de Ampère ∮ B · dl = μ₀ * I_enc
