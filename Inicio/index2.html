Flexbox

1. display: flex

¿Qué hace? Activa el modo Flexbox en un contenedor.
¿Para qué se usa? Se usa para convertir un contenedor en un contenedor flexible, haciendo que sus elementos hijos se distribuyan de manera flexible.
¿Cómo se usa? Simplemente se establece en el contenedor: 

display: flex;

9. align-self

¿Qué hace? Permite alinear un solo elemento flexible a lo largo del eje transversal.
¿Para qué se usa? Se usa para cambiar la alineación de un elemento individual dentro del contenedor.
¿Cómo se usa? Se asigna un valor de alineación: 

align-self: flex-end;

2. flex-direction

¿Qué hace? Define la dirección en la que los elementos flexibles son colocados en el contenedor.
¿Para qué se usa? Se usa para especificar si los elementos flexibles se colocan en fila, columna, o en direcciones inversas.
¿Cómo se usa? Valores comunes:

  • row (predeterminado): Los elementos se colocan en una fila.
  • row-reverse: Los elementos se colocan en una fila inversa.
  • column: Los elementos se colocan en una columna.
  • column-reverse: Los elementos se colocan en una columna inversa.
flex-direction: row;

3. justify-content

¿Qué hace? Define cómo se distribuyen los elementos flexibles a lo largo del eje principal.
¿Para qué se usa? Se usa para alinear elementos dentro del contenedor en el eje principal.
¿Cómo se usa? Valores comunes:

  • flex-start: Alinea los elementos al inicio del contenedor.
  • flex-end: Alinea los elementos al final del contenedor.
  • center: Alinea los elementos en el centro del contenedor.
  • space-between: Distribuye los elementos con el mismo espacio entre ellos.
  • space-around: Distribuye los elementos con espacio igual alrededor de ellos.
justify-content: space-between;

4. align-items

¿Qué hace? Define cómo se alinean los elementos a lo largo del eje transversal.
¿Para qué se usa? Se usa para controlar la alineación vertical de los elementos flexibles.
¿Cómo se usa? Valores comunes:

  • stretch (predeterminado): Estira los elementos para llenar el contenedor.
  • flex-start: Alinea los elementos al inicio del contenedor.
  • flex-end: Alinea los elementos al final del contenedor.
  • center: Alinea los elementos en el centro del contenedor.
align-items: center;

5. flex-wrap

¿Qué hace? Controla si los elementos flexibles se envuelven en líneas adicionales cuando no hay espacio suficiente.
¿Para qué se usa? Se usa para permitir que los elementos se distribuyan en múltiples líneas, lo que es útil cuando el contenido puede exceder el ancho del contenedor.
¿Cómo se usa? Valores comunes:

  • nowrap (predeterminado): Los elementos no se envuelven.
  • wrap: Los elementos se envuelven en líneas adicionales si es necesario.
  • wrap-reverse: Los elementos se envuelven en líneas adicionales en la dirección inversa.
flex-wrap: wrap;

6. flex-grow

¿Qué hace? Define cómo se expanden los elementos flexibles para llenar el espacio adicional.
¿Para qué se usa? Se usa para hacer que los elementos flexibles crezcan proporcionalmente para llenar el espacio disponible.
¿Cómo se usa? Se asigna un número que indica la proporción de crecimiento: 

flex-grow: 2;

7. flex-shrink

¿Para qué se usa? Se usa para que los elementos flexibles se encojan para adaptarse al contenedor.
¿Cómo se usa? Se asigna un número que indica la proporción de encogimiento:

8. flex-basis

¿Qué hace? Define el tamaño inicial de un elemento flexible antes de que se distribuya el espacio restante.
¿Para qué se usa? Se usa para establecer el tamaño de los elementos antes de aplicar flex-grow o flex-shrink.
¿Cómo se usa? Se puede establecer un tamaño fijo o un porcentaje: 

flex-basis: 100px;

CSS Grid

1. grid-template-columns

¿Qué hace? Define las columnas de la cuadrícula.
¿Para qué se usa? Se usa para especificar el número de columnas y su tamaño en un contenedor de CSS Grid.
¿Cómo se usa? Puedes usar valores fijos, porcentajes, o la unidad fr para establecer el tamaño: 

grid-template-columns: repeat(3, 1fr);

5. grid-template-areas

¿Qué hace? Define áreas nombradas en la cuadrícula.
¿Para qué se usa? Se usa para estructurar el diseño de la cuadrícula utilizando nombres de áreas.
¿Cómo se usa? Se define una cuadrícula con áreas específicas: 


                grid-template-areas:
                "header header header"
                "sidebar content content"
                "footer footer footer";

2. grid-template-rows

¿Qué hace? Define las filas de la cuadrícula.
¿Para qué se usa? Se usa para establecer la altura de las filas dentro de un contenedor de CSS Grid.
¿Cómo se usa? Similar a grid-template-columns, puedes establecer valores fijos o usar la unidad fr: 

grid-template-rows: 100px 200px;

3. gap

¿Qué hace? Define el espacio entre las filas y columnas de la cuadrícula.
¿Para qué se usa? Se usa para controlar el espacio entre los elementos de la cuadrícula sin usar márgenes o padding.
¿Cómo se usa? Puedes definir el espacio en píxeles, porcentaje, u otras unidades: 

gap: 20px;

¿Para qué se usa? Se usa para que un elemento abarque varias columnas o se posicione en una columna específica.
¿Cómo se usa? Puedes especificar el inicio y el final de la columna usando números o nombres de líneas:

8. justify-content

¿Qué hace? Alinea el contenido de la cuadrícula horizontalmente dentro del contenedor.
¿Para qué se usa? Se usa para controlar la distribución de las columnas dentro del contenedor de la cuadrícula.
¿Cómo se usa? Valores comunes:

  • start: Alinea las columnas al inicio del contenedor.
  • center: Alinea las columnas en el centro del contenedor.
  • end: Alinea las columnas al final del contenedor.
  • space-between: Distribuye las columnas uniformemente, con el primer elemento alineado al inicio y el último al final.
  • space-around: Distribuye las columnas uniformemente con espacio alrededor de cada columna.
  • space-evenly: Distribuye las columnas uniformemente con el mismo espacio entre ellas.
justify-content: space-between;
